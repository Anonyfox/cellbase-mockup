CSS


1. Bootstrap

Es können alle bekannten Bootstrap css Klassen verwendet werden. Voraussetzung dafür ist, dass nur Klassen der Bootstrap Version 2.3.2 verwendet werden. Die Referenz der Bootstrap Version 2.3.2 kann hier eingesehen werden. Ebenso steht dort ein Link zum Download der entsprechenden css und js Klassen bereit.

2. Buttons

Der Plague "Standard"-Button wird wie ein normaler Bootstrap Button eingebunden.

<button class="btn btn-plague" href="#">...</button>
Als zweiten Button gibt es einen Abbrechen Button der ensprechend dem Plague Design angepasst wurde. Dafür wird der Bootstrap Cancel Button nur geringfügig verändert.

<button class="btn btn-cancel" href="#">...</button>

3. Alignments

Es gibt ein paar Erleichterungen zum schnelleren Ausrichten von Content. Als erstes zu nennen ist die Klasse marge mit der man die Content-Breite auf 80% reduziert mit 10% Platz Links und 10% Platz Rechts.
Außerdem gibt es die drei Klassen img-float-left, img-float-right und img-float-none, welche dazu dienen den Textumfluss um zum Beispiel Bilder zu steuern.
